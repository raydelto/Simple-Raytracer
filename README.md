# Light & Geometry
## Raytracing using C++ Templates

Example Raytracing application, which supports:
* Multi-Sampling Anti-Aliasing
* Smooth Shadows
* Reflections

Written using C++ Templates.

Application can render:
* Sphere
* Cylinder
* Polygonal Mesh

Contains framework:
* Vector Math
* Ray Intersections
* Scene Graph
Features

    Raytracing
    Multi-Sampling Anti-Aliasing
    Smooth Shadows
    Reflections
    Sphere
    Cylinder
    Polygonal Mesh
    Vector Math
    Ray Intersections
    Scene Graph
